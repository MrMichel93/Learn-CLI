# Learn-CLI

Welcome to **Learn-CLI**! Master essential command-line skills through comprehensive tutorials and hands-on challenges. This repository is your complete guide to becoming proficient with the command-line interface.

## üìö Course Structure

This course is organized into four progressive modules, each in its own folder with detailed tutorials and mini challenges:

### [01 - Introduction to the CLI](./01-introduction/)
Get started with the command line and understand why it's such a powerful tool.
- What is the CLI and why learn it?
- Understanding the command prompt
- Your first commands
- **Mini Challenge:** Getting comfortable with the terminal

### [02 - Basic Commands](./02-basic-commands/)
Learn fundamental commands for navigating and managing files.
- Navigation: `pwd`, `ls`, `cd`
- File operations: `mkdir`, `touch`, `cp`, `mv`, `rm`
- Viewing files: `cat`, `less`, `head`, `tail`
- Wildcards and pattern matching
- **Mini Challenge:** File system navigation and management

### [03 - Intermediate Commands](./03-intermediate-commands/)
Build on your foundation with powerful text processing and system management tools.
- Text processing: `grep`, `find`, `sed`, `awk`, `sort`, `uniq`, `wc`
- File permissions: `chmod`, `chown`
- Process management: `ps`, `top`, `kill`, `jobs`
- Redirection and pipes
- Archiving: `tar`, `gzip`, `zip`
- Network commands: `ping`, `curl`, `wget`, `ssh`, `scp`
- **Mini Challenge:** Text processing and system management

### [04 - Advanced Commands](./04-advanced-commands/)
Master advanced techniques and automation for power users.
- Shell scripting: variables, conditionals, loops, functions
- Advanced text processing with regex
- System management: `systemctl`, `df`, `du`, `free`, `lsof`
- Advanced file operations: `rsync`, `ln`, `diff`
- Command-line productivity: `alias`, `history`, `xargs`
- Job control and shell features
- Text editors: `vim`, `nano`
- Environment variables
- Debugging and troubleshooting
- Package management
- Cron jobs and task scheduling
- Working with JSON using `jq`
- **Mini Challenge:** Building real-world automation scripts
